# labs/ — 独立プロトタイプ置き場

世の中のサーベイ系サービス（Wevox・Zonka Feedback・Delighted・Surveyapp等）のリサーチから
「コエに実装する価値がありそうな機能」を、**動くプロトタイプ**として試作したものです。

## ルール

- 各ファイルは**単一HTMLで完結**（本体の js/ css/ を一切参照しない・させない）
- デモデータ内蔵。バックエンド接続なし・外部送信なし
- ブラウザで直接開くだけで動く（GitHub Pages なら `…/labs/ファイル名.html`）
- 採用が決まったら docs/ROADMAP.md の対応WPとして**本体の作法で実装し直す**（コピペ移植はしない）
- 不要になったものは削除してよい

## 一覧

| ファイル | 何のプロトタイプか | 対応WP |
|---|---|---|
| `kiosk.html` | 待合室タブレット用キオスクモード。1画面1問・回答後も無操作でも自動リセット・本日の回答数表示 | WP-5 |
| `report.html` | A4・1枚の月次サマリーレポート（印刷用）。NPS・前月差・推移・設問別平均・自由記述抜粋 | WP-3 |
| `pulse.html` | 週1問のパルスサーベイ。10秒で回答→過去4週のミニ推移を回答者にも見せる | WP-9（要承認） |
| `text-insights.html` | 自由記述の頻出キーワード分析。クリックで該当コメント絞り込み。全てブラウザ内処理（外部送信なし） | WP-6 |
